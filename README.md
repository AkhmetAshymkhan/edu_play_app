# EduPlay — Educational Mobile App for Kids

EduPlay is an interactive educational mobile application designed for preschool children to learn the alphabet and numbers through gamification.

## Features
- Alphabet learning
- Number recognition
- Interactive mini-games
- Child-friendly UI/UX
- Audio and visual feedback
- Gamification elements

## Tech Stack
- Flutter
- Dart
- Firebase
- Figma

## Goal
The project was created as part of a Software Engineering study project focused on combining education and mobile technologies for children.
